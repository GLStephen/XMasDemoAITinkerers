# Capabilities

This section describes the key capabilities enabled by the AI Demo Project.

## Overview

The project provides three core capabilities:

1. **[Jira Integration](jira-integration.md)** - Automated ticket creation following project conventions
2. **[Confluence Sync](confluence-sync.md)** - Keep documentation current across platforms
3. **[AI Assistance](ai-assistance.md)** - Intelligent help throughout the development lifecycle

## How They Work Together

```
Development Workflow
====================

Planning          →  Execution         →  Communication
─────────────────────────────────────────────────────────
Epic specs        →  Jira tickets      →  Sprint demos
in markdown          auto-created         synced to Confluence

Technical docs    →  Implementation    →  Business summaries
maintained           guided by AI         auto-generated
```

## Benefits by Role

| Role | Primary Benefit |
|------|-----------------|
| Developer | Less time on ticket formatting |
| Product Manager | Consistent story structure |
| Tech Lead | Standardized conventions |
| Stakeholder | Always-current documentation |
| Leadership | Visibility without overhead |
