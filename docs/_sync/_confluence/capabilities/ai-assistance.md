# AI Assistance

## What It Does

Claude Code provides intelligent assistance throughout the development lifecycle - from planning to implementation to documentation.

## Business Value

- **Reduced overhead** - Automate repetitive tasks
- **Consistency** - Follow conventions without memorizing them
- **Knowledge capture** - Best practices encoded in skills
- **Faster onboarding** - New team members productive sooner

## Capabilities

### Planning Assistance
- Break down features into epics and stories
- Suggest acceptance criteria
- Identify dependencies and risks
- Estimate complexity

### Implementation Support
- Code generation and review
- Bug investigation
- Refactoring guidance
- Test creation

### Documentation
- Summarize technical content for business audiences
- Generate meeting notes from discussions
- Create demo scripts
- Update status reports

### Communication
- Translate between technical and business language
- Draft stakeholder updates
- Prepare demo narratives
- Answer project questions

## How It Works

Claude has access to:
- **Skills** - Project-specific conventions and procedures
- **Commands** - Reusable workflows (like Confluence sync)
- **Context** - Project documentation and history
- **Tools** - Jira, Confluence, and other integrations

This context allows Claude to provide relevant, project-aware assistance rather than generic responses.

## Best Practices

| Do | Don't |
|----|-------|
| Be specific in requests | Assume Claude knows everything |
| Review AI output | Accept without verification |
| Provide context | Skip important details |
| Iterate on results | Expect perfection first try |

## Getting Started

Simply ask Claude for help with any development task. The more context you provide, the better the assistance.
